# HabitLink 전체 기능 테스트 스크립트

## 📋 테스트 목적
이 스크립트는 HabitLink의 모든 분석 기능을 테스트하기 위해 작성되었습니다.
아래 스크립트를 자연스럽게 읽으면서 각 항목이 정확히 검출되는지 확인하세요.

---

## 🎯 검출 예상 항목

### 1. 특정 단어 반복
- **"이제"** - 3회 사용
- **"근데"** - 2회 사용

### 2. 비속어 사용
- **"시발"** - 1회 사용
- **"개새끼"** - 1회 사용 (문맥상 자연스럽게)

### 3. 문법 오류
- **"말한게"** → 올바름: "말한 게"
- **"빌어서"** → 올바름: "빌려서"
- **"되게"** → 올바름: "되게" (맥락에 따라 "정말" 권장)

### 4. 맥락 오류
- 날씨 → 축구 → **갑자기 코딩** (급격한 주제 전환)
- 프로젝트 얘기 → **갑자기 저녁 메뉴** (맥락 단절)

### 5. 말더듬
- **"저 저는"** - 단어 반복
- **"음... 어..."** - 필러 워드
- **"그-그니까"** - 음절 반복

## 📝 테스트 스크립트 (자연스럽게 읽으세요)

안녕하세요. **이제** 테스트를 시작해보겠습니다.

**저 저는** 김민준이라고 합니다. 오늘 HabitLink 시스템을 테스트하려고 하는데요.

**이제** 각 기능들이 제대로 작동하는지 확인해볼까요?

먼저 날씨 얘기부터 해볼게요. 오늘 날씨가 정말 좋네요. 하늘도 맑고 바람도 시원하고요.

**음...** **어...** 그럼 이번에는 다른 주제로 넘어가볼까요?

**근데** 어제 축구 경기 보셨어요? 정말 멋진 경기였어요. 손흥민 선수가 골을 **되게** 멋지게 넣었더라고요.

**그-그니까**, 제가 **말한게** 뭐였냐면요, 요즘 축구가 정말 재미있다는 거예요.

아, **이제** 갑자기 코딩하고 싶네요. 파이썬으로 뭔가 만들어보고 싶어요.

**근데** 진짜 **시발**, 코드가 자꾸 안 돌아가서 짜증나네요. 버그를 잡으려고 몇 시간을 썼는데도 안되니까요.

이 자리를 **빌어서** 한 가지 말씀드리고 싶은 게 있는데요, 프로그래밍할 때는 인내심이 정말 중요한 것 같아요.

요즘 제가 진행하고 있는 프로젝트가 있는데, 음성 분석 시스템이에요. 사용자의 말버릇을 분석해서 피드백을 주는 건데요.

아, 그런데 갑자기 오늘 저녁에 뭐 먹을까요? 치킨? 피자? 고민되네요.

프로젝트 얘기로 다시 돌아와서요, 이 시스템은 실시간으로 음성을 분석하고 문제점을 찾아내는 기능이 있어요.

그리고 그 **개새끼** 같은 버그를 드디어 잡았어요! 정말 뿌듯하더라고요.

테스트를 마치면서 느낀 점은, 이 시스템이 정말 유용할 것 같다는 생각이 들어요. 발표 연습이나 면접 준비할 때 도움이 많이 될 것 같습니다.

이상으로 테스트를 마치겠습니다. 감사합니다!

- [ ] "시발" 비속어 1회 검출
- [ ] "개새끼" 비속어 1회 검출
- [ ] 말더듬 검출 (저 저는, 음 어, 그-그니까)
- [ ] 발화 속도 측정 (정상 범위 확인)

### UI 시각화
- [ ] 키워드 검출 시 구가 연한 초록색으로 변경
- [ ] 비속어 검출 시 구가 연한 초록색으로 변경
- [ ] 오른쪽 상단에 검출 알림 표시
- [ ] 음성 크기에 따라 구 크기 변동
- [ ] 실시간 파형 표시

### 최종 PDF 보고서
- [ ] **Keyword Detection**: "이제" 3회, "근데" 2회
- [ ] **Profanity Detection**: "시발", "개새끼" 각 1회
- [ ] **Grammar Error Analysis**: 
  - "말한게" → "말한 게"
  - "빌어서" → "빌려서"
- [ ] **Context Error Analysis**:
  - 축구 → 코딩 (급격한 주제 전환)
  - 프로젝트 → 저녁 메뉴 (맥락 단절)
- [ ] **Stutter Analysis**:
  - Repetition: "저 저는"
  - Prolongation/Filler: "음... 어..."
  - Sound repetition: "그-그니까"
- [ ] **Speech Rate Analysis**: 평균 발화 속도 표시
- [ ] **Full Transcript**: 모든 발화 내용 타임스탬프와 함께 기록

## 💡 테스트 팁

1. **자연스럽게 읽기**: 로봇처럼 읽지 말고 자연스러운 억양으로 읽으세요.
2. **속도 조절**: 너무 빠르거나 느리지 않게, 평소 말하는 속도로 읽으세요.
3. **명확한 발음**: 특히 검출 대상 단어들은 명확하게 발음하세요.
4. **말더듬 연기**: "저 저는", "음... 어..." 부분은 실제로 더듬는 것처럼 읽으세요.
5. **적절한 쉼**: 문장 사이에 자연스러운 쉼을 두세요.

## 🔧 문제 발생 시

만약 특정 항목이 검출되지 않는다면:

1. **키워드**: `main.py`에서 "이제", "근데"를 키워드로 정확히 입력했는지 확인
2. **비속어**: `.data/profanity_list_ko.txt`에 해당 단어들이 있는지 확인
3. **문법/맥락**: LLM 분석이 활성화되어 있는지, API 키가 설정되어 있는지 확인
4. **말더듬**: 실시간 검출기와 사후 분석기 모두 활성화되어 있는지 확인
5. **타임스탬프**: 세션 시작 시간 기준으로 상대 시간이 제대로 표시되는지 확인
